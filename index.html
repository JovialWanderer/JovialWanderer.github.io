01 — Research

Areas of Interest

{ 01 }

Evolutionary Computation

Adaptive genetic algorithms for non-stationary financial environments. MAGA framework integrating warmstart scheduling, Adam-based operator control, and dynamic fitness sharing with theoretical analysis via the Price equation.

{ 02 }

Extreme Value Theory

Finite-sample error bounds for tail dependence estimators using Bernstein concentration and DKW inequalities. Optimal threshold selection, asymptotic normality via Lyapunov CLT, and applications to hydrological networks.

{ 03 }

Quantum Geometric Transport

Semiclassical wave-packet dynamics under inhomogeneous electric fields. Derived quantum metric-induced Bloch oscillations and transport signatures in tilted Dirac systems with vanishing Berry curvature.

Finite-Sample Bounds for Tail Dependence Estimators

Md. K. Faiyaz

Manuscript in preparation

Derives explicit finite-sample probability error bounds for nonparametric estimators of the upper tail dependence coefficient via Bernstein's inequality and DKW-based marginal estimation error. Establishes optimal threshold selection k* ≍ n^(2α/(2α+1)) and validates on Danube river network hydrology data.

Modeling Tail Dependence in Airline Stock Prices
